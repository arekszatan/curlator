### Everything tips and helps for project Curlator

  ```sh
  pyside6-uic name.ui -o name.py
  ```
  
* Do exe file for Python (pyw -> without display terminal, only gui)
  ```sh
  pyinstaller --onefile CurlatorMain.pyw 
  ```
